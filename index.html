The web is a collection of HTML documents. HTML = Hyper Text Markup Language - makes up basically every web page. Links between pages makes the web, weblike.

Major Pieces
Your Computer + Browser makes requests via the Internet to Servers using a protocol called HTTP. Servers respond with files that your browser displays. Servers are computers that host files.

HTML
Text Content is what you see.
Markup is what it looks like.
References to other documents such as images, videos etc.
Links to other pages

Markup is made up of TAGS
< Name > Contents < /Name >
opening tag --------- closing tag
Entirety is called "Element"

TAGS: (no spaces)
Bold = < b > < / b >
Italics/emphasis = < em > < / em >

Computers are "stupid." Code must be exact for it to work.

HTML Attributes

< tag attr = "value" > CONTENTS < / tag >

Anchors are tags for Links e.g. < a href = "www.reddit.com" > DERP < / a >
Image tags e.g. < img src = "url" alt = text > ; Text is what is displayed if image doesn't load.

Don't worry too much about rememerbing allthe details because there is too much to memorize; instead utilize google search if/when necessary'

White Space & Inline/Block tags
Inline elements are just in line with text. Block elements create an "invisible box."

Span and Div

Span - Inline; < span > text < / span>
Div - block; < div > text < / div >
Styles can be added to them to adjust how they are displayed via attributes such as CSS.
